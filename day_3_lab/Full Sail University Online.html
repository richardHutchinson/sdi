Lab 3: Following a Flowchart

( Due: Wed, 11 Sep | Status: Not Completed )

This lab is all about bringing together what you've learned over the last few days.  You'll create a project using variables, output, confirms, prompts, and conditionals.

Instructions

For this lab, you will implement a JavaScript version of this flowchart:

A few things you'll want to note:

  • You must have an example of a Boolean conditional, a string conditional, and a number conditional, but they do not need to be in the order they show in the flowchart.  You can do the string or number conditional first.
  • The prompt that comes before the nubmer conditional will require you to parse the input as a float or integer before the conditional.
  • You can use the story idea from the Day 1 LAb if you like or you can come up with a completely different story.
When complete, upload the files to your GitHub account.

Deliverables

Compress your project folder and rename your archive using the lastname_firstname_lab3 naming convention before uploading. Do not upload an unzipped file or you will get a zero for the assignment.
