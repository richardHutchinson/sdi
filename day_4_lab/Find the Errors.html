Find the Errors

The ability to debug and fix code is just as important as being able to write code. This activity will help you hone your debugging skills.

Level of Effort

This activity should take approximately 30m to complete. It will require:

  • 0m Research
  • 0m Prep & Delivery
  • 30m Work

If you find that this activity takes you significantly less or more time than this estimate, please contact me for guidance.

Instructions

The first thing you'll need to do is download the sdi-lab4-errors.zip file attached to this activity. Extract the files in the sdi-lab4-errors.zip file and open them in your code editor. You'll also want to load the HTML file in your browser and open the console.

Hint: When correct, the console should tell the story of a cat wrangler and the job that he does.

Once you've corrected all the errors, compress all the files into a zip called lastname_firstname_lab4errors.zip and upload it to FSO.
