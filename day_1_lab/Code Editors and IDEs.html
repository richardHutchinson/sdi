Code Editors and IDEs

This activity gives you information about a variety of code editors available to you. By the end of this activity, you should be ready to select a code editor that you can use throughout this class.

Objectives & Outcomes

Successful completion of this activity should include the following:
  • Understand the difference and similarities of various code editors.
  • Select a code editor to use during this class.

Instructions

Code Editors and Evironments for Client Side Development

As developers, there are few pieces of software we will spend as much time with as code editors. What follows is a survey of eight of the most popular client-side editors for the Mac platform.

While there are many code editors out there that support client-side as well as application and server-side development, we've focused on editors and IDEs (interactive development environments) marketing themselves for HTML, CSS, and JavaScript design and development.

Text Editors & IDEs

TextWrangler (Bare Bones) This light-weight "little brother" app to the more extensive BBEdit IDE, TextWrangler's simple color coding, built in text-transformations and FTP support have made it the go-to for many developers who want to make a quick change.
Sublime Text - Version 2 is in beta which means that you can use it as long as you don't mind the occasional "don't forget to buy it" pop up alert. It is not a full-featured application like many of the others listed here, but many enjoy the minimalism and speed of the light-weight editor. The introspective auto completion and innovative scrolling preview side bar are personal favorites.
Komodo Edit (Active State) This application boasts a "Fast Open" feature which is attractive to developers wanting to get a quick look at a file.
Espresso (MacRabbit) The selection of plug-ins for Espresso, cleverly called sugars, provide extensibility, but fall short of the plug-in libraries available for Coda and Eclipse-based editors like Aptana. I particularly like the CSS3 support and the DOM viewer incorporated in the view. Keep in mind that this is software that is maintained by a group of hardworking students, so updates are less than frequent.
WebStorm (Jet Brains) A lighter, quicker version than the Java IDE IntelliJIDEA that is built specifically for JavaScript developers. The auto-complete and error spotting is one of the most comprehensive of the applications listed here. And the cherry on top is that it is free for educational purposes!
Coda (Panic) A favorite IDE among mac web developers, Coda just released its version 2.0 which brings in many features other editors already have and adds features like Git tie-ins (in addition to SubVersion) for managing source code.
Dreamweaver (Adobe) Like other Adobe products, the Dreamweaver IDE has developed a reputation for slowness and bloat. It's also worth noting that its WYSWIG (what you see is what you get) capabilities it has earned a reputation for being the non-developers development tool.The CS6 version does boast a nice addition to its mobile web support and tie-in with Phone Gap for producing phone applications.
Aptana Studio -(Appcelerator) Athough this IDE is one of the newer ones on this list, Aptana is already very popular among web developers. Because it is a build of the open-source Eclipse engine, users have access to an extensive plugin library and also inherit the Eclipse's slow speed. It also comes with some very nice ftp and git source control integration.
