Revision Control and Git

Create an account at Github.com, and set up the application on your Mac. This is the technology you will use to submit your work in this course.

Objectives & Outcomes

Successful completion of this activity will show that you can do the following:
  • Create a version cntrol repository
  • Understand how to use Git to commit files and changes to a repository

Level of Effort

This activity should take approximately 90m to complete. It will require:

  • 70m Research
  • 0m Prep & Delivery
  • 20m Work

If you find that this activity takes you significantly less or more time than this estimate, please contact me for guidance.

Instructions

Watch the videos included with this activity to understand what Git is and how it is used in a development environment. Then, follow the instructions in the set-up videos to set up your own GitHub.com account. Unless otherwise noted in an activity, you will submit all of your assignments via your Git repository.

GitHub currently alloows you to create unlimited free public accounts. This is what you should do for this class. But, you are also required to submit a request for a free private account available to students by going to https://github.com/edu. If your private account is activated in time for this class, you can use this account for assignment submission by adding your instructor to the account.
