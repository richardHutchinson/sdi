Problem 1

Create a function that will accept a string as and argument and then check to see if the string follows the pattern of an email address, such as aaa@bbb.ccc. Keep in mind that all you need to check is that the string contains only one @ symbol and no spaces. There is no need to check for .com, .edu, etc. The result should be output to the console in a meaningful way.

Problem 2

Create a function that will accept three strings. The first will be a list of items separated by a given string, the second will be that string separator, and the third will be a new separator. The function will then perfrom the task of replacing the first separator with the second and return the result. Thus, a list such as "a,b,c,d" with the first separator being "," and the second separator being "/" would become "a/b/c/d". Again, be sure to output the result in a meaningful way.
