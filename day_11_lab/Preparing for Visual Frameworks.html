Instructions

Titanium is a development tool that you can use to create mobile apps using JavaScript. You'll be using it in the next several coding classes, so it's important to have the tool set up and ready to go by the time you arrive in VFW. Some of you may have already installed this tool in a previous class. If not, this activity will give you the guide you need to begin working with Titanium.

  1. Go to the Appcelerator Web site and click on Products. This will give you a list of all the products Appcelerator has to offer.
  2. Click Titanium.
  3. Click the Download Titanium button. This will take you to a page where you can sign up for an account. You must do this in order to downlaod the development tool.
  4. Fill in the necessary information, and click the Sign Up button. If you already have an account, click the link next to the button to log into your account.
  5. Click the link for the Titanium Studio for Mac. This will download the dmg file.
  6. Install the dmg file.

If you have not yet installed Xcode, you will be asked to do so. Similarly, if you do not have the Android SDK installed, you will need to do that as well before you can begin developing. The configuration requirements change frequently, so check the Titanium Web siter for the latest information on what is needed.
