Problem 1 - Format a number

Create a function that will format a number to a specified number of decimal places. The function will need to accept two numbers: a number to be formatted and a number that represented decimal places. It should return the newly formatted number. Don't forget to output the number in a meaningful way.

Problem 2 - Number vs. String

As you have learned, there is a difference between a number as a string data type and a number as a Number data type. For this task, you will simply need to send a string into the function and return it as a Number. If you want to get creative, you also can add a conditional to determine if the string sent into the function is a number before you do the conversion.

Problem 3 - Date Differences

Create a function that will find the difference between two dates. This function will require three parameters. The ﬁrst two will be dates that you want to compare, and the third will be a string to determine if the function returns hours or days. The function will then perform the math necessary to ﬁnd the difference between the two dates and return the choice as hours or days. Again, remember to output the returned value in a meaningful way.
